Then you are going to copy following code in <link rel = "stylesheet" type = "text/css" href = "myStyle.css">. You are going to change the file address into myStyle.css. CSS Syntax | Video 7


Image above was created by W3Schools

As you can see, each CSS codes was made up by 2 big parts: Selector and Declaration

Let's talk about selector first, There are generally 4 types of selector. They are:

  • Universal Selector: This selector select everything on the webpage. For selector, you will add * .
  • Tag Selector: This is most commonly used selector, you just type the tag name that you want to get selected, and you are done!
  • Class/ID Selector: If you want to style certain class or id, it also fine! For class, add . in front of your class name as your selector. For id, add # in front of your id name as your selector.
  • Nested Selector: If you want to select a div tag inside of a div tag, you will be using div > div as your selector.

What is inside of Declaration? Well, 2 things: property and value.

For example if I want to change all paragraph tag to color orange, color will be the property and orange will be the value.

Properties

Let's dig in deeper to discover properties and values for flexbox! You are welcome to paste the code into a code editor to see the effect!

flex-direction

flex-direction: column The column value stacks the flex items vertically (from top to bottom)

flex-direction: column-reverse The column-reverse value stacks the flex items vertically (but from bottom to top)

flex-direction: row The row value stacks the flex items horizontally (from left to right)

flex-direction: row-reverse The row-reverse value stacks the flex items horizontally (and you guessed it, from right to left)

flex-wrap

flex-wrap: wrap The wrap value specifies that the flex items will wrap if necessary.

flex-wrap: nowrap The nowrap value specifies that the flex items will not wrap (default).

flex-wrap: wrap-reverse The wrap-reverse value specified that the flexible item will wrap if neccessary, in reverse order.

flex-flow property

flex-flow: row wrap This is shorthand for setting both flex-direction and flex-wrap properties.

justify-content

justify-content: center The center value aligns the flex items the center of the container.

justify-content: flex-start The flex-start value aligns the flex items at the beginning of the container (this is default)

justify-content: flex-end The flex-end value aligns the flex items at the end of the container

justify-content: space around and justify-content: space-between.

Psuedo-"stuff"

It's time to make your website a little bit more interative, we will use psuedo-classes and psuedo-elements to achieve that goal.

What is pseudo-classes?

A pseudo-class is a selector that selects elements that are in a specific state, e.g. they are the first element of their type, or they are being hovered over by the mouse pointer. They tend to act as if you had applied a class to some part of your document, often helping you cut down on excess classes in your markup, and giving you more flexible, maintainable code.

Example of Psuedo-classes

format of the psuedo-class:


selector:psuedo-class

:first-child This pseudo-class always target the first child element in the article.

:last-child It did opposite what last psuedo-class did, it always target the last child element in the article.

:only-child This pseudo-class always target the only given child element in the article.

:invalid Checks the form input when it's invalid, such as it required an email address but user typed random numbers.

:hoverThis only applies if the user moves their pointer over an element, typically a link.

:focusonly applies if the user focuses the element using keyboard controls.

How to configuring the animation?

You will need to create an animation property inside of your CSS selector. You will also need to configure additional properties inside of @keyframes. Here are the lists for the sub-properties of the animation property:

  • animation-nameSpecifies the name of the @keyframes at-rule describing the animation’s keyframes.
  • animation-durationConfigures the length of time that an animation should take to complete one cycle.
  • animation-timing-functionConfigures the timing of the animation; that is, how the animation transitions through keyframes, by establishing acceleration curves.
  • animation-delayConfigures the delay between the time the element is loaded and the beginning of the animation sequence.
  • animation-iteration-countConfigures the number of times the animation should repeat; you can specify infinite to repeat the animation indefinitely.
  • animation-directionConfigures whether or not the animation should alternate direction on each run through the sequence or reset to the start point and repeat itself.
  • animation-fill-modeConfigures what values are applied by the animation before and after it is executing.
