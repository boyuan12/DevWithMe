Intro to CSS | Video 5

So. CSS

Today we are going to discover features in CSS, so we can make our website more stylish. CSS is stands for Cascading Style Sheet

Where?

So where do we add CSS codes? There are three options you can choose:

  1. Added straight in style tag - This way is most convenient way to do, but in organization, it also can make your HTML code hard to find. So what you will basically do, is to add <style> tag inside of <head> tag. Then write your css codes and properties inside of <style> tag
  2. Use inline style (I forgot to mention this in the video). you will add style="Put your CSS Properties here" at the end of each tag but inside of >.
  3. Or using external script - so you are going to create external CSS file, remember using .css as extention. Then you are going to copy following code in <link rel = "stylesheet" type = "text/css" href = "myStyle.css">. You are going to change the file address into myStyle.css. You can use online script if you want, which I will go over how to set them up in next video.

Some properties:

background-color: is using to change background color of certain tag. You CAN'T use this for "normal" tags (such as header tags, paragraph tags), you must use div or span tag. font-size: is used to specify how large you want the text to be, remember, No Naked Number, which means you must include units at the end, such as px, em, vp, we will get to them later. font-family: is used to specify which font you want to use. There are some default font that browser can understand, such as sans-serif, serif, monospace, arial, etc.
However, sometime these default fonts won't be enough, that where Google Fonts come in handy. Go to fonts.google.com, then choose the font you want to use, copy the font's codes into your CSS file or to your HTML file header. color: this is use to change the text color of certain text. P.S.
