What is HTML?

  1. HTML stands for Hyper Text Markup Language.
  2. HTML helps to made up the ACTUAL contents in the web.
  3. HTML made up by tags
  4. HTML tags usually use "< >" to saperate from each oher

What is CSS

  1. CSS stands for Cascading Style Sheet
  2. CSS helps make up the style of the webpage
  3. CSS codes/properties are made up by { }

First, in order to access code editor, you have a few choices. Here are some recommendations for the code editor software:

The following codes are standard layout:

<!DOCTYPE html>
<html>
 <head>
 </head>
 <body>
 </body>
</html>

We also learned some tags in HTML:

  • <h1 ~ h6> These are for header tags, h1 is the largest, and h6 is the smallest
  • <p> This tag is for paragraph
  • <a href = ""> This tag help you to show a clickable link to another webpage
    • <img src =""> This is how you would show an image in yout webpage <button> This tag shows button <ul> This is unordered list, or bulleted list <ol> This is ordered list, or numbered list <li> This tag is stands for line item, it will need to include in ol or ul in order to actually show the items <br> This create a line break <!-- --> This create comments. The text inside of this tags will not rendered by browser, instead, it's for human to understand your codes better.

Format your text

<b> Bold text <strong> Important text (it technically does the same thing as <b>) <i> Emphasized text (it technically does the same thing as <i>) <mark> Marked text <small> Small text <del> Deleted text <ins> Inserted text <sub> Subscript text <sup> Supscript text

Inline element

<div> div is really useful especially when you start using CSS and JS. <span> span is useful when you dealing with inline element.

ID and Classes

You can create id, it looks something like this <div id="id"> Something you need to keep in mind, you can't use number as your first character, and id must be unique. You will learn why in next video

You can also create class, it looks something like this <div class="class">
