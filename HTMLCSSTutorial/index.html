  • <h1 ~ h6> These are for header tags, h1 is the largest, and h6 is the smallest

    • <img src =""> This is how you would show an image in yout webpage <button> This tag shows button <ul> This is unordered list, or bulleted list <ol> This is ordered list, or numbered list <li> This tag is stands for line item, it will need to include in ol or ul in order to actually show the items <br> This create a line break <!-- --> This create comments. Format your text

<b> Bold text <strong> Important text (it technically does the same thing as bold) <i> Italic text <em> Emphasized text (it technically does the same thing as italic) <mark> Marked text <small> Small text <del> Deleted text <ins> Inserted text <sub> Subscript text <sup> Supscript text

Inline element

<div> div is really useful especially when you start using CSS and JS. <span> span is useful when you dealing with inline element.

Form in HTML | Video 4

We will learn how to create form elements, such as text field, radio button, checkboxes, with placeholder and required

<input type="text"> This created a text field. <label for="language"><input id="language" type="checkbox" name="language"> HTML</label> This created a checkboxes. Label declared what is these checkboxes for, id is for specific style to specific checkbox, and name can differentiate between checkboxes in same group. <label for="experience"><input id="experience" type="radio" name="experience">No</label> This create radio button, it's actually nearly as same as checkbox, except type is radio. <button type="submit">this button submits the form</button> This create a submit button.

How can I required an option? Add required at the end of the option that you want to let that to be required to fill in.

How can I add pre-selected item? Add check at the end of the checkbox or radio button you want to selected on.
