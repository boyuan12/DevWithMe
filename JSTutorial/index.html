Variable

Variable is important concept in every programming language, no exception on JavaScript. There are 3 ways to declare variables:

  • varThis is the oldest way to declare variable in JavaScript, you can reassign the value of this variable.
  • constThis is one of new version of JavaScript variables. This declare a constant variable, which means the value is not reassignable.
  • letThis is one of new version of JavaScript variable. This declare a variable that cannot reassign under "normal" condition, however, you can reassign this variable inside of a loop, or not "normal" environment.

Name a variable

In order to name a variable, your variable name must follow:

  • First letter MUST be a letter or _
  • Must ONLY use letter, number and _
  • It is case sensitive
  • No length limitation
  • No reserve words usage

A little bit on operator

The + operator in JS has 2 usage, one is add two or more strings together, second is to actually add number, just like in regular arithmetic.

function vs function (syntax)

In algebra, you learned a function is something like this: f(x) = 2x + 6, where f means it's a function, x is an input, 2x + 6 is the rule of that function. JS function is similar, to create a function, you will type:

function function_name() { };

The style of function's name are same as variable's style. When you name a function, make sure your function's name matches that result, or output of that function.

Conditional Statement 1 | Video 6

Real-world example...

You may have heard of this statement before: "If you review for your test effectively, then you may get better score. Otherwise you will receive a lower score on your test."

Conditional Statement is exactly just that, it's execute different code based on the conditions. But before we actually create some codes, I want to clear some terminology first.

= == ===

= means assign in variable.


== comparison of equal to (doesn't have to be same type)

=== comparison of equal to (must be the same type)

If...

First statement you will learn is if statement, it only work if the condition is true, copy and paste the codes to the console here:

const a = 0; if (a === 0) {console.log("a is 0");}

You should see a is 0 as output.

If... else ...

This kind of condition also works when if is not true, it's going to execute else. Take a look on following codes:

var reviewTest === true; if (reviewTest === true) {console.log("Good score");} else {console.log("Bad Score");}

You should saw "Good score" as output.

if... if .. else .. else ...

This is called nested code. Basically, you add another condition inside of if part. Take a look of this code for example:

var reviewTest === true; var efficient = false; if (reviewTest === true) {if (effcient === true){console.log("Good"); } else {console.log("Bad");}} else {console.log("Bad Score");}

You should see "bad" as return

if ... else if... else

I didn't mention this in the video, but what this does is if the if statement is false, then it's going to execute else if, if this return false, it will execute else. Try to create a statement on your own.
