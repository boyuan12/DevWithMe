Why JS?

If you have any previous programming experience before, most of them need either need to compile your program, need specific-interpreter or need download library files. Well, JavaScript is non of those. You don't need to do any of these, all you need is click command-option-J/control-shift-J (at chrome) to open up your browser's JavaScript console, and you can start type your JS codes there.

What can you do with JS?

Because it can make your webpage more interactive, it can create interactive games, and it can create natural simulations.

Variable

Variable is important concept in every programming language, no exception on JavaScript. There are 3 ways to declare variables:

  • varThis is the oldest way to declare variable in JavaScript, you can reassign the value of this variable.
  • constThis is one of new version of JavaScript variables. This declare a constant variable, which means the value is not reassignable.
  • letThis is one of new version of JavaScript variable. This declare a variable that cannot reassign under "normal" condition, however, you can reassign this variable inside of a loop, or not "normal" environment.
